# 媒体库管理

Kaloscope 面向本地媒体文件管理场景，媒体库是核心功能之一。

## 功能范围

根据 README，当前媒体库能力至少包含：

- 支持电影、电视剧等媒体库类型
- 实时监控文件系统，自动识别新添加的媒体文件
- 从 NFO 文件中自动提取并解析元数据

## 典型流程

1. 创建媒体库并指定根目录。
2. 配置文件监控或手动触发扫描。
3. 系统识别新文件并尝试关联元数据。
4. 用户在界面中浏览、筛选、播放或继续整理。

## 建议的目录组织

虽然项目文档中尚未给出完整规范，但对媒体库通常建议：

- 电影与剧集分开管理
- 下载临时目录与最终入库目录分离
- 命名规范尽量统一，便于匹配与补刮
