# 播放器与用户系统

## 在线播放

根据 README，Kaloscope 内置播放器并支持以下能力：

- FLV、HLS、MP4 格式播放
- 弹幕显示
- 移动端横屏播放
- 播放进度记录与续播

这意味着项目不仅负责资源获取和入库，也覆盖了媒体消费环节。

## 用户与权限

README 中提到系统支持：

- 多用户
- 管理员与普通用户角色区分
- 基于媒体库和索引器的访问权限分配
- 个人偏好设置与头像自定义

## PWA 与国际化

作为 Web 应用，项目还支持：

- 以 PWA 方式安装到桌面或移动设备
- 自动生成多尺寸图标
- 简体中文与英文界面
