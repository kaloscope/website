# 项目简介

Kaloscope 是一款以可视化工作流为核心的本地媒体库管理工具。

与传统媒体管理工具不同，Kaloscope 不把资源搜索、元数据刮削、详情解析等逻辑硬编码到程序内部，而是将这些步骤封装为可编辑、可组合的工作流节点。这样一来，你可以针对不同站点、不同数据源和不同自动化需求，自行调整流程。

## 核心理念

### 工作流优先

搜索、详情抓取、下载计划、自动任务等能力，优先通过工作流表达。程序本身负责提供运行时、调度、权限与界面，而站点接入细节由工作流决定。

### 本地媒体中心

项目围绕本地媒体文件管理展开，强调媒体库整理、下载管理、播放体验和可持续维护的工作区结构。

### 开放扩展

项目支持从 GitHub 仓库导入社区工作流模板，也允许用户自行编写工作流、脚本和适配配置，用于扩展站点支持范围。

## 功能概览

- 可视化工作流编辑器
- 多索引器资源搜索
- 媒体文件扫描与 NFO 元数据解析
- aria2、qBittorrent、Transmission 等下载器支持
- 内置播放器，支持 FLV、HLS、MP4
- 多用户与权限控制
- PWA 安装与国际化界面

## 适用场景

- 家庭媒体服务器或 NAS 环境
- 需要自动搜索、下载和整理番剧资源的个人用户
- 想自行接入私有站点或定制刮削逻辑的高级用户

## 非目标与注意事项

项目 README 已明确说明：

- 本项目仅供个人学习与技术交流使用
- 社区或第三方工作流可能包含任意代码或网络请求
- 使用者应自行审查工作流的安全性、合法性和行为边界
